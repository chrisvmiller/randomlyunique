Hasta la Palindrome

May 02, 2016 

As a tech worker, I can unequivocally state that the only thing more important than debating coding styles or following Agile methodologies is the ability to rapidly whiteboard algorithms. So, if you're ever asked to write a palindrome detector during a tech screen, be sure to impress the interviewer with this beauty, written in ArnoldC:

IT'S SHOWTIME
  HEY CHRISTMAS TREE potentialPalindrome
    YOU SET US UP 12321

  HEY CHRISTMAS TREE isPalindrome
    YOU SET US UP @I LIED
  GET YOUR ASS TO MARS isPalindrome
  DO IT NOW checkPalindrome potentialPalindrome

  BECAUSE I'M GOING TO SAY PLEASE isPalindrome
      TALK TO THE HAND "It is a Palindrome"
  BULLSHIT
      TALK TO THE HAND "It is NOT a Palindrome"
  YOU HAVE NO RESPECT FOR LOGIC
YOU HAVE BEEN TERMINATED

LISTEN TO ME VERY CAREFULLY checkPalindrome
  I NEED YOUR CLOTHES YOUR BOOTS AND YOUR MOTORCYCLE number
  GIVE THESE PEOPLE AIR

  HEY CHRISTMAS TREE temp
    YOU SET US UP number
  HEY CHRISTMAS TREE reverse
    YOU SET US UP 0
  HEY CHRISTMAS TREE tempMod10
    YOU SET US UP 0
  HEY CHRISTMAS TREE isPalindrome
    YOU SET US UP 0

  STICK AROUND temp
      GET TO THE CHOPPER tempMod10
        HERE IS MY INVITATION temp
        I LET HIM GO 10
      ENOUGH TALK

      GET TO THE CHOPPER reverse
        HERE IS MY INVITATION reverse
        YOU'RE FIRED 10
        GET UP tempMod10
      ENOUGH TALK

      GET TO THE CHOPPER temp
        HERE IS MY INVITATION temp
        HE HAD TO SPLIT 10
      ENOUGH TALK
  CHILL

  GET TO THE CHOPPER isPalindrome
    HERE IS MY INVITATION number
    YOU ARE NOT YOU YOU ARE ME reverse
  ENOUGH TALK

  I'LL BE BACK isPalindrome
HASTA LA VISTA, BABY
