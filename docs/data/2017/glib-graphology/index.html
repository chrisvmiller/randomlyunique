Glib Graphology

Mar 17, 2017  data

Graphology, the act of evaluating personality characteristics from handwriting, isn't too objective, but - hey - I'll still take a crack at it! Using a set of 300 famous signatures, I'll attempt to compose my personality from various celebrities!

The algorithm in a nutshell involved calculating image keypoints via a Scale-Invariant Feature Transform (SIFT), followed by a simple pixel-wise Manhattan distance comparison between my keypoints (it's actually a small normalized neighbourhood around these keypoint) and each celeb.
