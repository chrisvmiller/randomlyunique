Lost Lagoon Solitaire

Jun 05, 2022 

After about a month of work, my wife and I created a solitaire collection game using Godot 3. The games included are Black Hole Solitaire, Calculation Solitaire, FreeCell Solitaire, and Spider Solitaire. We also modified each game to use a simple two-suit deck, which increases the win rate.
